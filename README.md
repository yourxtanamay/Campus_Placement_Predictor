# Campus Placement Predictor 🎓💼

## Project Overview

The **Campus Placement Predictor** is a Machine Learning project that predicts whether a student will get placed or not based on academic performance and other factors. The model analyzes different attributes such as secondary education marks, higher secondary marks, degree percentage, work experience, and specialization.

This project helps demonstrate how **data analysis, visualization, and machine learning** can be used to solve real-world problems in the education and recruitment domain.

---

## Features

* Data preprocessing and cleaning
* Handling missing values
* Exploratory Data Analysis (EDA)
* Data visualization using graphs
* Machine Learning model training
* Placement prediction
* Simple **HTML interface** for displaying results

---

## Dataset

The dataset used in this project contains information about students such as:

* Secondary School Percentage (SSC)
* Higher Secondary Percentage (HSC)
* Degree Percentage
* Specialization
* Work Experience
* Employability Test Score
* Placement Status

The dataset helps train the machine learning model to predict placement outcomes.

## How to Run the Project

1. Download or clone the repository.
2. Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

3. Open the Jupyter Notebook:

```bash
jupyter notebook
```

4. Run the notebook **Campus_Placement_Predictor_MV.ipynb**.

5. Open the **HTML file** to view the interface.
